    About FNLookup

    Information

    FNLookup is a page hosted on GitHub Pages, a free page hosting system for GitHub.

    FNLookup is provided by Fortnite APIs that provide information from the game.

    Personal Information

    FNLookup does NOT collect any information about the user, however Fortnite APIs collect the country of residence of the user for analytics.

    Here are any types of information that is NOT collected by FNLookup

    • IP Address
    • Browser type
    • Device
    • Epic Games account credentials

    Information may or may not be as well collected by GitHub's free page hosting service: GitHub Pages.

    Saved Data

    Only data that is stored locally is your request language.

    Languages are independent from each machine and do not carry on with Google Accounts or any other type of account.

    Deleting your data

    You have an option to delete your data: Right below with this button. This will delete everything that is saved currently, like your language.

      FNLookup

      FNLookup is a easy browsing tool for Fortnite related content.

      Features of this page

      1. Fortnite content
        1. Battle pass
        2. Map and map comparisons
        3. Season Progress
        4. Augments
        5. News
      2. Other
        1. Player Stats
        2. Seasons
        3. Twitch Drops
      3. Cosmetics and Item Shop
        1. Predictions
        2. Search cosmetics
        3. View cosmetics
        4. See the newest added cosmetics
        5. See the Item Shop
      4. Multilingual API Requests
